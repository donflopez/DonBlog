Welcome to DonBlog.

En este proyecto voy a realizar una plataforma completa para tener mi propio blog y que cualquiera pueda utilizarla para tener el suyo. DonBlog constará de una instalación sencilla y podrá ser fácilmente editado.

Qué necesitas para instalarlo?

  • Node.js instalado.
  • NPM instalado.
  • MongoDB instalado.

Una vez tengas todos los requisitos, clona el proyecto y ejecuta:


$ [sudo] npm install
$ node app.js

Durante el inicio, el sistema comprobará si es la primera vez que se ejecuta y (de ser así) creará en la base de datos todo lo necesario para funcionar.
